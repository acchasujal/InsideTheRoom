# User Journey: The 3-Minute Demo Flow

## Overview
This journey is optimized for the 3-minute judging demo. It prioritizes recall, emotional impact, and the "Experience Before Explanation" mechanic.

## Stage 1: The Hook (0:00 - 0:10)
- **Action:** The user enters the experience.
- **Narrative:** "Football's biggest controversies aren't about what happened. They're about what a word means."
- **Goal:** Immediately reframe the user's expectation from a "data analytics" tool to a "legal linguistics" experience.

## Stage 2: The Incident Cycles

### Cycle 1: Perišić Handball (0:10 - 0:45)
- **Setup:** User views the 2018 World Cup Final handball incident.
- **Decision 1:** User decides if it's a penalty.
- **Perspectives:** User sees conflicting AI arguments (e.g., Fan vs. Player).
- **Reveal:** "Deliberately" has no definition. "Unnaturally bigger" has no definition.
- **Decision 2:** User realizes their certainty was built on undefined words.

### Cycle 1b: VAR Nested Layer (0:45 - 1:05)
- **Setup:** Focus shifts to the VAR review of the Perišić handball.
- **Reveal:** The phrase "clear and obvious error" has no definition.
- **Reflection:** The correction mechanism has the same problem as the foul itself.

### Cycle 2: De Jong Chest Kick (1:05 - 1:45)
- **Setup:** User views the 2010 World Cup Final chest kick.
- **Decision 1:** User chooses between no foul, yellow, or red.
- **Reveal:** Howard Webb gave yellow, later said red. Same referee, same footage, two answers. "Reckless" vs. "excessive force" has no defined boundary.
- **Decision 2:** User confronts the impossibility of objective consistency.

### Cycle 3: Mbappé Offside (1:45 - 2:00) *[Optional/Cut if short on time]*
- **Setup:** Quick overview of the "deliberate play" offside reset.
- **Reveal:** "Deliberately played" and its guidance terms are undefined. Undefined all the way down.
- **Decision 2:** User acknowledges recursive ambiguity.

## Stage 3: The Climax & Close (2:00 - 3:00)

### Cycle 4: Suárez Goal-Line Handball (2:00 - 2:45)
- **Transition:** "The first three incidents showed terms the law doesn't define. This is different."
- **Setup:** User views the 2010 incident.
- **Decision 1:** "Was justice served?"
- **Reveal:** The law doesn't define 'justice' or 'adequate remedy', only procedure. Ghana still went home.

### The Close (2:45 - 3:00)
- **Action:** Final thesis statement delivered.
- **Narrative:** "Football law doesn't need better data. It was designed to need better judgment. And it put that judgment in a room where the people making the call are using words that have no definitions. That's what we built."
- **Goal:** Leave the judge with a single, highly memorable insight proving Granite's value.

## Stage 4: Live Generation (If Time Permits)
- **Action:** User is invited to input a novel incident (Layer 2).
- **Goal:** Prove Granite is structurally necessary by generating live, divergent perspectives that a static site cannot.
